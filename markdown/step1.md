# 힌트
### RestAssured
- 미리 제공한 StationAcceptanceTest 코드를 활용하세요 :)

### given
- 요청을 위한 값을 설정 (header, content type 등)
- body가 있는 경우 body 값을 설정 함
### when
- 요청의 url와 method를 설정
### then
- 응답의 결과를 관리
- response를 추출하거나 response 값을 검증할 수 있음
